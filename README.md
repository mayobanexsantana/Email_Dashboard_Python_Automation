  <legend align="left"><h3>About Repository</h3></legend>

  <em>
    <p>Many Data Analyst are requested to monitor some indicators and to report through email. This may imply taking the data from a database,
    generate charts and other type of dashboard, put it on and email and send with some analysis to a list of intersted people.
    The report generation only can take long time and it will be better to use that time for analysis instead of building the same
    dashboard structure everytime.</p>
    <p>One script will is designed to run a SQL query to an existing ODBC connection. It is also assumed that the machine running
    the script has Outlook configured with an account; OS expected windows.</p>
    <p>The other script is designed to run a SQL query to an existing ODBC connection. You are not expecting Outlook configuration, 
      but you have user address (user@domain.com) and SMTP IP address.</p>
